# Apresentação
Esse é um projeto template das minhas aplicações em node.js que utiliza Express e MongoDB.

# NPMs
npm init -y

package.json
incluir: "type": "module"

npm i nodemon
package.json > incluir: em 'scripts > "dev": "nodemon server.js"

npm i express

npm i mongodb
npm i mongoose
